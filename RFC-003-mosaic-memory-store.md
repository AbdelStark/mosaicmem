# RFC-003: Mosaic Memory Store
3D patch storage with spatial indexing. insert_keyframe(): VAE-encode frame → split into patches → unproject to 3D → store with provenance. retrieve(): given target camera pose → frustum cull point cloud → for each visible point look up source patch → reproject to target view → score by visibility/overlap/distance → select top-K → return MosaicFrame. compose_mosaic(): flatten patches into token sequence for attention injection. Memory manipulation: splice_horizontal (side-by-side scenes), flip_vertical (Inception-style), erase_region (scene editing), insert_external (add objects).
