# RFC-008: Autoregressive Pipeline
Chained multi-window generation for 2+ minute rollouts. Per window: (1) get camera poses, (2) retrieve memory mosaic, (3) compose conditioning (text + temporal context + spatial memory + PRoPE), (4) run DiT denoising loop, (5) decode via VAE, (6) update memory store with new keyframes, (7) yield frames. Keyframe selection: every Kth frame or significant camera motion threshold. Memory budget limits total stored patches. Chained Forward Training (training-only, reference): chain windows during training to create larger pose variations, forcing model to use spatial memory for consistency.
